The knowledge was acquired from the SCC0216 - Computational Modeling with Graphs - ICMC/USP taught by Prof. Maria Cristina Ferreira de Oliveira

#### 1. Adjacency Matrix
#### 2. BFS (Breadth First Search)
#### 3. DFS (Depth First Search)
